Automated validation from this site

This repo runs Playwright journey tests on GitHub Actions: they load this site from the same URLs visitors use, open the hosted StreaKit demo in a new tab and run a short record / freeze / unfreeze pass with streak and state assertions, then follow the Unloop showcase and Projects links to the public macOS zip, verify SHA-256 against downloads/manifest.json, and list the archive to confirm it contains Unloop.app as in Testing downloads. On every push, tests run against the workspace copy; on a schedule (and when you run the workflow manually) they run against https://aryankk.github.io as well.

Public test docs. Unloop distribution repo: Testing downloads, developer test setup. StreaKit dev kit: CI builds the monorepo; @streakit/core ships Vitest unit tests under packages/core/src/__tests__ (no Playwright in that public repo yet). Full Unloop desktop journey Playwright suites run against a local Tauri dev server in private source, not in Unloop-Application, which is intentionally distribution-only.
